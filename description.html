Описание управления игрой SWAT

Для управления своим персонажем используется компьютерная мышь, путем перемещения курсора по игровоу полю.

Для выстрела необходимо направить шар персонажа в сторону вражеского юнита и нажать на кнопку мыши

Игровой персонаж обозначен белым цветом.

Вражеские юниты представляют собой шары обозначенные красным цветом.

Враги движутся в сторону игрового персонажа.

Скорость движения со временем увеличивается.

Если хотя бы один вражеский юнит коснется игрового персонажа игра будет проиграна.

Ниже приведены минимальные системные требования к ПК:

  • ОЗУ 512 МБ
  • Видеокарта 128 МБ
  • Поддержка JavaScript браузером
